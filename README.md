# 1分钱喂猫咪 - 抖音直播互动小玩法

一款挂载在抖音直播间的互动小玩法。观众通过送直播间礼物来"喂"虚拟猫咪，积累公共进度条，进度条满后触发喂食动画和全屏特效。

## 项目概述

### 技术栈
- **游戏引擎**: Cocos Creator 3.x (TypeScript)
- **猫咪动画**: Spine 2D 骨骼动画
- **直播互动**: 抖音直播小玩法 SDK
- **数据持久化**: localStorage (本地存储)

### 核心功能
- 🐱 **3只猫咪角色**: 赛博帅猫、可怜猫、像素淘气猫
- 🎁 **8种礼物互动**: 支持选猫和喂食两种功能
- 📊 **公共进度条**: 所有观众共享，满值触发喂食动画
- ⭐ **等级进化系统**: 3个进化阶段（Lv1 / Lv5 / Lv10）
- ✨ **全屏特效**: 匹配猫咪风格的粒子特效
- 💾 **本地存储**: 猫咪等级/经验值持久化

## 猫咪设计

### 赛博帅猫 (Cyber Cat)
- **画风**: 赛博朋克风 - 霓虹光效、电路纹理、全息护目镜、深蓝紫色系
- **性格**: 高冷、不屑、偶尔傲娇
- **选中礼物**: 能力药丸 (10抖币)
- **喂食反应**: 嫌弃地闻一下 → 勉强吃 → 推了一下护目镜

### 可怜猫 (Pitiful Cat)
- **画风**: 日系治愈风 - 水彩质感、大眼泪汪汪、柔和粉色调
- **性格**: 楚楚可怜、感恩、爱哭
- **选中礼物**: 甜甜圈 (52抖币)
- **喂食反应**: 先呜咽 → 看到食物眼睛发亮 → 吃完后开心打滚

### 像素淘气猫 (Pixel Cat)
- **画风**: 16-bit像素风 - 复古游戏感、明亮绿橙色系
- **性格**: 调皮、贪吃、会偷食物
- **选中礼物**: 棒棒糖 (1抖币)
- **喂食反应**: 先假装不感兴趣 → 突然抢过去 → 吃完打嗝冒出像素气泡

## 游戏机制

### 礼物系统

| 礼物 | 抖币价格 | 功能 | 进度值 |
|------|---------|------|--------|
| 仙女棒 | 1 | 喂食 | +5 |
| 棒棒糖 | 1 | 选猫（像素淘气猫）+ 喂食 | +5 |
| 大啤酒 | 1 | 喂食 | +5 |
| 人气票 | 1 | 喂食 | +5 |
| 仙女棒花束 | 10 | 喂食 | +40 |
| 能力药丸 | 10 | 选猫（赛博帅猫）+ 喂食 | +40 |
| 甜甜圈 | 52 | 选猫（可怜猫）+ 喂食 | +180 |
| 礼花筒 | 99 | 喂食 | +320 |

### 进度条机制
- **公共进度条**: 所有观众共享，满值为 1000
- **填充速度**: 约 20-30 个小礼物或 3-5 个大礼物可喂满一轮
- **满值触发**:
  1. 猫咪进入 eating 状态，播放喂食反应动画（3-5秒）
  2. 全屏特效播放（匹配猫咪风格的粒子效果）
  3. 经验值增加，判定是否升级
  4. 如果升级，播放进化动画（额外 2-3秒）
  5. 进度条重置，回到 idle 状态

### 经验与等级
- 每次成功喂食获得经验值：基础 10 点 + 本轮进度值/100（向下取整）
- 等级阈值：Lv1→Lv5（50经验），Lv5→Lv10（200经验）
- 等级数据本地持久化（localStorage）

### 猫咪状态机
每只猫有 4 种状态：
1. **idle**（待机）：轻微呼吸/摇尾巴循环动画
2. **excited**（兴奋）：收到礼物时，期待表现
3. **eating**（进食）：进度条满，播放喂食动画
4. **evolved**（进化）：经验值达到阈值，升级特效

## 核心模块

### GameManager
游戏主循环，初始化和协调各模块，管理全局游戏状态。

### GiftHandler
接收 SDK 礼物事件，解析礼物类型和来源用户，分发给 CatManager 或 ProgressManager。

### CatManager
管理 3 只猫的切换逻辑和状态机，维护当前选中猫咪，管理等级和进化。

### ProgressManager
进度值累加计算、进度条 UI 更新、满值判定和触发回调。

### EffectManager
全屏特效的创建和播放、贡献气泡管理、粒子效果生命周期管理。

## 发布流程

1. 抖音开放平台注册（个人身份，手机号/银行卡实名认证）
2. 创建"直播小玩法"应用
3. 使用开发者工具 + 抖音直播伴侣本地调试
4. 配置礼物互动信息（8种礼物 + 效果描述）
5. 提交包体审核（需上线说明文档 + 版本描述）
6. 审核通过后在直播伴侣中挂载使用

## MVP 范围

### 包含
- 3 只猫咪（各 1 个基础外观 + 4 种状态动画）
- 等级系统（3 个阶段，本地存储）
- 公共进度条 + 8 种礼物映射
- 全屏喂食特效（3 套，匹配猫咪风格）
- 贡献气泡显示
- 猫咪状态机（idle/excited/eating/evolved）
- 选猫机制（3 种礼物同时选猫+喂食）
- 进度条 >80% 猫咪急切表现

### 不包含（后续版本）
- 个人进度条
- 观众排行榜 / 贡献榜
- 云端数据同步
- 更多猫咪（>3只）
- 猫咪皮肤商城 / 装扮系统
- 观众任务系统
- 直播 PK 模式
