# 喵星食堂 - 猫咪喂食互动游戏

一款基于 TypeScript 开发的猫咪喂食互动游戏。玩家通过赠送礼物来喂养虚拟猫咪，积累公共进度条，进度条满后触发喂食动画和全屏特效。

## 功能特性

- 🐱 **3 只猫咪角色**：赛博帅猫、可怜猫、像素淘气猫，各具风格
- 🎁 **8 种礼物互动**：支持选猫和喂食两种功能
- 📊 **公共进度条**：所有玩家共享，满值触发喂食动画
- ⭐ **等级进化系统**：3 个进化阶段（Lv1 / Lv5 / Lv10）
- ✨ **全屏特效**：匹配猫咪风格的粒子特效
- 🔊 **音效系统**：Web Audio API 实现的互动音效
- 💾 **本地存储**：猫咪等级/经验值持久化
- ⌨️ **键盘快捷键**：支持快速操作

## 技术栈

- **语言**：TypeScript
- **前端**：原生 HTML / CSS / JavaScript
- **特效**：CSS 动画 + DOM 粒子系统
- **音效**：Web Audio API
- **数据持久化**：localStorage

## 猫咪介绍

### 赛博帅猫
- **画风**：赛博朋克风 - 霓虹光效、蓝紫色系
- **性格**：高冷、不屑、偶尔傲娇
- **选中礼物**：能力药丸

### 可怜猫
- **画风**：日系治愈风 - 水彩质感、柔和粉色调
- **性格**：楚楚可怜、感恩、爱哭
- **选中礼物**：甜甜圈

### 像素淘气猫
- **画风**：16-bit 像素风 - 复古游戏感、绿橙色系
- **性格**：调皮、贪吃、会偷食物
- **选中礼物**：棒棒糖

## 游戏机制

### 礼物系统

| 礼物 | 价格（抖币） | 功能 | 进度值 |
|------|-------------|------|--------|
| 仙女棒 | 1 | 喂食 | +5 |
| 棒棒糖 | 1 | 选猫（像素淘气猫）+ 喂食 | +5 |
| 大啤酒 | 1 | 喂食 | +5 |
| 人气票 | 1 | 喂食 | +5 |
| 仙女棒花束 | 10 | 喂食 | +40 |
| 能力药丸 | 10 | 选猫（赛博帅猫）+ 喂食 | +40 |
| 甜甜圈 | 52 | 选猫（可怜猫）+ 喂食 | +180 |
| 礼花筒 | 99 | 喂食 | +320 |

### 进度条机制
- **公共进度条**：所有玩家共享，满值为 1000
- **满值触发**：
  1. 猫咪进入 eating 状态，播放喂食反应动画
  2. 全屏特效播放（匹配猫咪风格的粒子效果）
  3. 经验值增加，判定是否升级
  4. 如果升级，播放进化动画
  5. 进度条重置，回到 idle 状态
- **急切状态**：进度 ≥ 80% 时，猫咪进入急切状态

### 经验与等级
- 每次成功喂食获得经验值：基础 10 点 + 本轮进度值/100（向下取整）
- 等级阈值：Lv1→Lv5（50 经验），Lv5→Lv10（200 经验）
- 等级数据本地持久化（localStorage）

### 猫咪状态机
每只猫有 5 种状态：
1. **idle**（待机）：轻微呼吸循环动画
2. **eager**（急切）：进度 ≥ 80% 时的期待表现
3. **excited**（兴奋）：收到礼物时的期待表现
4. **eating**（进食）：进度条满，播放喂食动画
5. **evolved**（进化）：经验值达到阈值，升级特效

## 核心模块

### GameManager
游戏主循环，初始化和协调各模块，管理全局游戏状态。

### GiftHandler
接收礼物事件，解析礼物类型和来源用户，分发给 CatManager 或 ProgressManager。

### CatManager
管理 3 只猫的切换逻辑和状态机，维护当前选中猫咪，管理等级和进化。

### ProgressManager
进度值累加计算、进度条 UI 更新、满值判定和触发回调。

### EffectManager
全屏特效的创建和播放、贡献气泡管理、粒子效果生命周期管理。

### StorageManager
本地存储管理，负责猫咪数据的持久化保存和加载。
